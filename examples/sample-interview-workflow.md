# 脱敏示例：从原始片段到访谈稿

这个示例是虚构的，只用于展示工作流。人物、项目、数字和场景均为脱敏样例。

## 1. 原始片段

```text
采访者 00:00
你当时为什么决定离开那份稳定工作？

嘉宾 00:08
其实不是突然决定的。我大概从前一年 10 月就开始想走了，但那时候没钱，所以先攒钱。我算了一下，如果每个月只花最基本的钱，够我撑 8 个月。我当时没有贷款，父母身体也还可以，所以我觉得可以试一次。

嘉宾 00:42
真正让我受不了的不是某一个领导，而是我发现那条路再走 5 年，还是不会变成我想要的生活。我以前也想过升职，后来发现那个体系里需要的不是我擅长的东西。

采访者 01:15
离开后你马上开始做项目了吗？

嘉宾 01:20
没有。我先去了两个城市住了一段时间。那时候其实挺迷茫的，想赚钱，但没有方法。快结束的时候，卡里只剩几万块，我才真的慌了。

嘉宾 01:55
后来我做过一个很小的本地项目，第一周赚了 6000 块。钱不算特别多，但那次特别重要，因为我第一次觉得，原来我不回去上班也能拿到钱。
```

## 3. 人物变化链条

```text
想要的生活：不再被稳定工作绑定，希望拥有更自主的时间和选择。
起点限制：没有明确赚钱方法，只能先靠存款争取试错时间。
第一次尝试：离职前先攒钱，计算最低生活成本和家庭压力。
失败或代价：离开后并没有马上找到项目，旅居快结束时开始焦虑。
微小正反馈：本地项目第一周赚 6000 元。
获得的能力或信心：第一次确认自己可以不靠工资拿到钱。
再次尝试：继续寻找能放大的项目。
可见成果：后续可围绕流量、产品和交付建立个人商业闭环。
成为现在的自己：从稳定系统里的一个岗位，开始把自己当成一家公司经营。
```

## 4. 主题大纲

### 一句话主题

一个从稳定工作离开的人，如何通过算账、试错和一次小正反馈，开始把自己从“一个岗位”变成“一家公司”。

### 章节

1. **离开不是冲动，是先算清楚自己能不能试错**
   - 来源：S01
   - 作用：建立可信的离职前提，不把故事写成热血裸辞。

2. **真正的问题不是领导，而是那条路通不到想要的生活**
   - 来源：S02
   - 作用：写清楚人物欲望和路径不匹配。

3. **自由之后，真正的压力才出现：想赚钱，但没有方法**
   - 来源：S03
   - 作用：保留迷茫和现实压力，避免直接跳到成功。

4. **第一周 6000 元不大，但它让他放弃回去上班的念头**
   - 来源：S04
   - 作用：用微小正反馈解释人物为什么能继续。

## 5. 问答稿片段

**采访者：**  
你当时为什么敢离开那份稳定工作？

**嘉宾：**  
我不是突然决定的。大概从前一年 10 月开始，我就知道自己想走了，但那时候第一反应不是马上辞职，而是先攒钱。

我算过一笔账：如果每个月只花最基本的钱，我的存款够撑 8 个月。当时我没有贷款，父母身体也还可以，不需要我立刻承担很大的家庭压力。算完之后，我才觉得，自己可以拿出一段时间试一次。

所以它不是一个很热血的决定。更像是我先确认，自己有一个小小的安全垫，然后才敢从原来的轨道里出来。

**采访者：**  
真正让你想走的是什么？

**嘉宾：**  
不是某一个领导，也不是某一次委屈。

我后来想明白的是，那条路再走 5 年，还是不会变成我想要的生活。我以前也想过升职，想在体系里做得更好，但慢慢发现，那个体系需要的东西不是我擅长的，也不是我想长期交换的。

这个判断出来之后，离开就不是“受不了了”，而是我知道自己要换一条路。

**采访者：**  
离开之后顺利吗？

**嘉宾：**  
不顺利。我先去了两个城市住了一段时间，那段时间其实挺迷茫的。你会觉得自己终于自由了，但很快会遇到另一个问题：我想赚钱，可是我没有方法。

真正开始慌，是旅居快结束的时候。卡里只剩几万块，那个时候我才意识到，光体验生活是不够的，我必须真正钻进一个项目里。

后来我做了一个很小的本地项目，第一周赚了 6000 块。钱不算特别多，但对我很重要。那是我第一次觉得，原来我不回去上班，也能从市场里拿到钱。
```
